Specimen Sheet.

Island PT · Design System v0.1 · Brand register

Audit · anti-patterns this system corrects

  1. #000 / #fff reflex. Replaced with tinted ink + bone in OKLCH.
  2. Identical card grids. Three intentionally distinct card species, never lined up uniformly.
  3. Hero-metric template. Three hero variants, none with "big number + small label" strips.
  4. Em-dash overload. All system copy uses commas, colons, periods.
  5. Uniform radii. Four radius tokens — flat, soft, pill, arc — applied per component class.
  6. Italic-serif tic. Italic accents are now scoped (display headings, brand lockup) not sprinkled per heading.
  7. Flat rhythm. Six gap tokens; no component shares the same vertical breathing room by default.
  8. Restrained-by-reflex palette. Coral and sun-gold promoted to named roles; drenched variant added.
  9. No form tokens. Inputs, selects, textareas, checkboxes, hover/focus/error states are all defined.
  10. !important cascades. No !important anywhere in this file.
  11. Hardcoded gallery grid. Three reusable gallery layouts: asymmetric, strip, hero+thumbs.
  12. No motion language. Single ease-out-quart curve, two durations — fast (140) and mid (240).
